## 📊 Overview
This project analyzes financial performance and identifies the root cause of a significant profit margin decline in 2024.

## 🎯 Key Findings
- Profit margin dropped from ~35% to ~17% starting July 2024
- Revenue remained stable, indicating the issue is cost-driven
- Cost ratio increased from ~65% to ~82–83%
- The issue affects all products and customer segments

## 💡 Business Recommendations
- Investigate key cost drivers (operations, suppliers, infrastructure)
- Optimize cost structure across product lines
- Focus on high-revenue segments for cost efficiency
- Consider pricing adjustments if needed

## 🛠 Tools Used
- BigQuery (SQL)
- Tableau
- Data Analysis & KPI Modeling
